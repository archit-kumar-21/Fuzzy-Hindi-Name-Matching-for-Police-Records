# 🔍 Fuzzy Name Matching System

🚀 Built as a Data Science + Web Integration Project

A Flask-based web application that detects and matches similar or duplicate names using fuzzy matching techniques. This system helps in identifying duplicate records, improving data quality, and enabling efficient data cleaning in real-world scenarios.

---

## 📌 Features

* 🔎 Fuzzy matching of names using similarity scores
* 📊 Detects duplicate or closely related records
* 🌐 Interactive web interface built with Flask
* 📁 Upload and process datasets
* ⚡ Fast and efficient matching using optimized algorithms

## ⚙️ Installation

1️⃣ Clone the repository:

```
git clone https://github.com/your-username/fuzzy-name-matching-system.git
cd fuzzy-name-matching-system
```

2️⃣ Create a virtual environment:

```
python -m venv venv
venv\Scripts\activate      # For Windows
source venv/bin/activate   # For Linux/Mac
```

3️⃣ Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```
python app.py
```

Now open your browser and go to:

```
http://127.0.0.1:5000
```

## 🧠 How It Works

* Uses fuzzy string matching techniques (like Levenshtein Distance)
* Compares input names with dataset records
* Assigns similarity scores
* Identifies potential duplicates or similar entries
